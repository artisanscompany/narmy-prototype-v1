Design Decisions

  • Gold accent line on the card top — subtle premium touch over the reference
  • Rounded square shield icon instead of circle — more modern, aligns with shadcn's design language
  • Gradient sign-in button — depth and polish over flat green
  • Subtle grid pattern on background — military precision aesthetic without being heavy
  • Monospace font for input placeholders — reinforces the "system identifier" feel
  • Error state will shake the card gently + show red inline message below fields
  • Successful login triggers a brief loading spinner, then redirects based on role
